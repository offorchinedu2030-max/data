Project Overview
This Power BI dashboard analyzes the distribution and delivery of military equipment to law enforcement agencies. It provides interactive visualizations for monitoring equipment allocation, delivery status, inventory trends, and agency-level performance

Objectives
- Track equipment deliveries
- Monitor delivery status
- Analyze distribution by agency and equipment type
- Generate operational insights

-  Dashboard Features
- KPI Cards
- Delivery Status Analysis
- Equipment Distribution
- Agency Comparison
- Interactive Slicers

- Tools
- Microsoft Power BI
- Power Query
- DAX
- Data Modeling

-  Business Value
Supports logistics monitoring, accountability, and data-driven decision-making.
